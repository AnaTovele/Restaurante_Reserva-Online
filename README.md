##  Sobre o Projeto

O site do **Sabor & Arte** inclui:

- Página inicial com banner, destaques e apresentação  
- Página "Sobre Nós" com missão, visão, valores e equipa  
- Página de reservas com formulário estilizado  
- Página de contacto  
- Layout totalmente responsivo para desktop, tablet e mobile



##  Tecnologias Utilizadas

- **HTML5** — Estrutura das páginas  
- **CSS3** — Estilização e responsividade  
- **JavaScript** — Interatividade leve  
 - **GitHub Pages** — Hospedagem gratuita para front-end

## Como o Projeto Foi Criado

1. **Planeamento**
   - Definição da identidade visual e layout  
   - Organização das páginas e seções

2. **Estruturação do HTML**
   - Navbar e rodapé comuns  
   - Seções de banner, cards, equipa e formulário

3. **Estilização com CSS**
   - Cores, tipografia, espaçamentos e responsividade  
   - Grid para cartões e secções

4. **Interatividade**
   - Botões, animações leves e scroll suave  
   - Validação de formulários simples (JS opcional)

5. **Testes**
   - Verificação da responsividade em vários dispositivos  


##  Como Testar Localmente

###  Requisitos
- Navegador moderno (Chrome, Firefox, Edge, Safari)  
- Editor de código (VS Code recomendado)  
- Servidor local PHP (opcional, caso use formulários)  

###  Passo a Passo

1. **Clonar o repositório**

git clone https://github.com/AnaTovele/Restaurante_Reserva-Online


Abrir a pasta do projeto

Restaurante_Reserva-Online

Abrir localmente

Clique duas vezes em index.html ou

Abra no navegador a partir do VS Code (Live Server recomendado)

 ## Hospedagem no GitHub Pages

Suba todos os arquivos para o repositório do GitHub:

HTML, CSS, imagens e scripts

Vá até Settings → Pages

Configure:

Source: Deploy from a branch

Branch: main

Folder: / (root)

Salve e aguarde alguns segundos.
O GitHub irá gerar o link do site:

https://SEU-USUARIO.github.io/SEU-REPOSITORIO/
